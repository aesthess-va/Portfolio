Il progetto

Set ambient per DJ ØSC al Morg Voltapagina di Messina. Palette eterea: rosa, azzurro, bianco. Il flusso visivo è pensato per respirare insieme alla musica — lento, ipnotico, organico.

Approccio tecnico: come base generativa ho integrato una patch TouchDesigner della community, un operatore scritto in codice da un utente che produceva effetti di distorsione organica sui video. Ho registrato i risultati tramite movie file out, costruendo un archivio di clip originali. Il VJ set è stato eseguito con la stessa interfaccia TD usata alla Balza, mixando queste clip con immagini di natura selezionate da fonti free.
